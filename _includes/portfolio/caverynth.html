CAVERYNTH

DOWNLOAD & PLAY ONLINE: ITCH.IO

Short 2D platformer originally made for the CovidGameJam in April 2020. The theme of the jam was repetition
At the time of making this game I was (and probably still am) obsessed with the game Celeste. I wanted to make a platformer that, like Celeste, featured levels challenging both the skill plane and the problem-solving plane.

The game features 12 levels progressively built over the previous one. Starting from a simple walk, in each level a new obstacle is added to the course.
This creates the feeling of repeating the same level, while progressively adding challenge, either via skill checks in which the player needs to be precise, or "puzzle" obstacles, which require thinking to find the path to the end of the level.

I took this game as an excellent chance of exploring 2D level design, and also fiddled with the new Unity 2D lighting system and the particle system.

Tools used: Unity & Aseprite The game is playable from a browser or downloadable for Windows.
