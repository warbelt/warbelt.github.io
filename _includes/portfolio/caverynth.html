CAVERYNTH

DOWNLOAD & PLAY ONLINE: ITCH.IO

Short 2D skill-based platformer. At the time of making this game I was obsessed with the game Celeste. I wanted to make a platformer that, like Celeste, featured levels challenging both the player's mechanical andproblem-solving skills.

The game features 12 levels all built over the previous one. Starting from a simple walk, in each level a new obstacle is added to the course.
This creates the feeling of repeating the same level, while progressively adding challenge, either via skill checks in which the player needs to be precise, or "puzzle" obstacles, which require thinking to find the path to the end of the level.

This game was an excellent chance for exploring 2D level design, and playing with the new Unity 2D lighting system and the particle system.

Caverynth acquired some popularity in GameJolt.com, occupying one of the top positions in the Platformer - Hot category. It was also one of the games selected to be featured in Cuarentena Perpetua, an online showcase of Spanish videogames ran by IndieMad.
